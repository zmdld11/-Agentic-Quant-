# 🤖 Agentic Quant QQ机器人使用手册

欢迎使用 **Agentic Quant (AI 智能量化投研)** QQ机器人！
本机器人在后台接入了深度定制的量化分析框架与大语言模型（如 DeepSeek），能够结合最新的A股市场数据、公司公告与全球宏观政治新闻，为你提供“极客风”的单日个股走势推演。

## 🌟 核心功能

- **一键个股诊断**：在QQ群聊或私聊中输入指令，机器人即可实时爬取目标股票的走势、基本面与新闻，并返回长篇深度政经推演研报。
- **群聊分享属性**：将研究报告直接发在群里，方便跟群友们一起探讨背后的逻辑。
- **零门槛即开即用**：作为使用者，你或你的朋友只需要在QQ群里发消息即可，所有爬虫和AI推理都在开发者后台无感完成。

## 💬 如何使用指令

与机器人交互非常简单。无论是私聊还是在开启了机器人的群聊中，都支持以下指令：

### 📝 1. 个股深度推演 (查询/分析某个股票)
你可以直接艾特(`@`)机器人，或者使用特定的触发词（如 `/分析` 或 `查股票`）。

**指令示例：**
- `@机器人 查股票 600519`
- `/分析 002594`
- `@机器人 帮我看看这只股票：600036` 

**机器人工作流程与耗时提示：**
收到指令后，由于步骤众多，大约需要等待 **10 ~ 20 秒**。
因为机器人在这短短几秒内，正在做这些事：
1. 抓取该A股今日最新的K线状态与主营业务。
2. 抓取与该股票相关的今日专属重大新闻。
3. 抓取全网今日宏观政经重点新闻。
4. 将所有万字资料浓缩并丢给深度学习大模型（DeepSeek）进行多维度逻辑推断。
*(提示：如果群内多人同时查询，系统可能会排队，请耐心等待，切勿频繁刷屏。)*

### 📊 2. 帮助菜单 (查询基本操作)
忘记了怎么用？直接呼叫帮助！

**指令示例：**
- `@机器人 帮助`
- `/help`
机器人会返回包含所有可用指令的使用指南。

## 🛠️ （开发者专用）如何搭建并运行本机器人

如果你是群主或开发者，想要在你的电脑或者服务器上跑起这个机器人，请参考以下整体思路：

### 1. 技术栈构成
- **底层量化框架**：本项目的 `AgenticQuant` 引擎。
- **机器人应用层**：基于超高人气的 Python 框架 [NoneBot2](https://nonebot.dev/)。
- **QQ通信协议侧**：使用目前非常稳定防封的 **NapCatQQ** (无头客户端) 基于 `OneBot v11` 与框架进行通信。

### 2. 第 1 步：安装机器人环境
在项目当前虚拟环境中，安装所需的 NoneBot2 和协议适配器：
```bash
pip install nonebot2[fastapi] nonebot-adapter-onebot nonebot2[httpx]
```

### 3. 第 2 步：配置 NapCatQQ
下载并启动 `NapCatQQ` 客户端，建议用准备当机器人的 QQ 小号扫码登录，并在其 WebUI 后台中配置反向 WebSocket，指向你的 NoneBot2 服务地址（通常默认是 `ws://127.0.0.1:8080/onebot/v11/ws`）。

### 4. 第 3 步：编写与运行主程序
编写带有 `@on_command` 装饰器的 QQ 机器人入口脚本（替代现有的 `main.py`），然后在控制台启动：
```bash
python bot.py
```
